Análisis de Seguridad Vial

Este proyecto fue desarrollado para la Agencia de Seguridad Vial con el objetivo de mejorar la seguridad en zonas críticas de tránsito. Se centra en el análisis de incidentes ocurridos en avenidas y esquinas, puntos donde se concentra la mayor frecuencia de accidentes.

A través de un proceso de análisis exploratorio de datos (EDA), limpieza de datos y visualización interactiva con Power BI, se identificaron patrones de riesgo, áreas de mayor incidencia y variables asociadas a los incidentes viales.

👉 ¿Querés explorarlo? Descargá el archivo desde el repositorio y abrilo en Power BI para navegar las visualizaciones interactivas. Además, el análisis exploratorio se desarrolló con Python y herramientas como Pandas, Seaborn, Matplotlib, Scikit-learn y Excel.

Tecnologías: Power BI, Excel, Python, Pandas, Scikit-learn

Foro Atractivo y Responsivo

Proyecto Integrador | Desarrollo Web Full Stack

Desarrollo de un foro dinámico y visualmente atractivo, diseñado para ofrecer una experiencia de usuario fluida y responsiva en todo tipo de dispositivos. El objetivo principal es permitir a los usuarios crear y compartir publicaciones de forma intuitiva, con soporte para imágenes, fechas de publicación y una interfaz moderna.

🛠️ Tecnologías Utilizadas

  • Backend: Node.js, Express, Sequelize, MySQL
  • Frontend: EJS (Embedded JavaScript Templates)
  • Otras herramientas: dotenv, cors, morgan, mysql2

🚀 Funcionalidades Destacadas

  • Creación de publicaciones con título, contenido, imagen (URL) y fecha.
  • Visualización organizada de los posts con imágenes optimizadas.
  • Diseño completamente responsivo y adaptable a distintos dispositivos.
  • Integración con base de datos MySQL utilizando Sequelize como ORM.

📂 Instalación y Ejecución

  • Crear la base de datos forodb y la tabla publicaciones según el esquema indicado.
  • Renombrar el archivo example.env a .env y configurar las variables necesarias.
  • Instalar las dependencias con: npm i
  • Ejecutar la aplicación: node app.js
  • Acceder desde el navegador: http://localhost:4000

Análisis para el Sector Gastronómico

Intelidata es una solución analítica integral desarrollada para transformar el sector gastronómico mediante el análisis inteligente de datos. El foco está en la ciudad de Los Ángeles, California, ayudando a inversores del reubro gastronómico a tomar decisiones basadas en evidencia.

🚀 ¿Qué resuelve este proyecto?

  • Comprender patrones de comportamiento del cliente.
  • Detectar oportunidades de inversión en zonas estratégicas.
  • Optimizar decisiones estratégicas con modelos predictivos.

🧠 Habilidades demostradas:

  • ✅ Análisis Exploratorio (EDA) de datos no estructurados.
  • ✅ Procesamiento de Lenguaje Natural (NLP) de reseñas.
  • ✅ Modelado predictivo con Machine Learning.
  • ✅ Visualización con Looker Studio y Streamlit.
  • ✅ Trabajo en equipo ágil con documentación técnica.

🔍 Lo que encontrarás:

  • 📌 Diagnóstico del mercado gastronómico en LA.
  • 📌 Insights de satisfacción del cliente a partir de opiniones.
  • 📌 Recomendaciones accionables basadas en IA.
  • 📌 Dashboards interactivos en Streamlit y Looker.

🛠️ Tecnologías:
Python, Pandas, NumPy, Scikit-learn, NLTK, Matplotlib, Seaborn, Looker Studio, Streamlit, GitHub

🎯 Resultado: una herramienta inteligente para diagnosticar el mercado, medir reputación online, detectar oportunidades y diseñar estrategias de marketing efectivas.
