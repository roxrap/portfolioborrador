Análisis de Seguridad Vial

Este proyecto fue desarrollado para la Agencia de Seguridad Vial con el objetivo de mejorar la seguridad en zonas críticas de tránsito. Se centra en el análisis de incidentes ocurridos en avenidas y esquinas, puntos donde se concentra la mayor frecuencia de accidentes.

A través de un proceso de análisis exploratorio de datos (EDA), limpieza de datos y visualización interactiva con Power BI, se identificaron patrones de riesgo, áreas de mayor incidencia y variables asociadas a los incidentes viales.

👉 ¿Querés explorarlo? Descargá el archivo desde el repositorio y abrilo en Power BI para navegar las visualizaciones interactivas. Además, el análisis exploratorio se desarrolló con Python y herramientas como Pandas, Seaborn, Matplotlib, Scikit-learn y Excel.

Tecnologías: Power BI, Excel, Python, Pandas, Scikit-learn
