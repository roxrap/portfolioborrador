Análisis de Seguridad Vial

Este proyecto fue desarrollado para la Agencia de Seguridad Vial con el objetivo de mejorar la seguridad en zonas críticas de tránsito. Se centra en el análisis de incidentes ocurridos en avenidas y esquinas, puntos donde se concentra la mayor frecuencia de accidentes.

A través de un proceso de análisis exploratorio de datos (EDA), limpieza de datos y visualización interactiva con Power BI, se identificaron patrones de riesgo, áreas de mayor incidencia y variables asociadas a los incidentes viales.

El proyecto propone recomendaciones concretas basadas en evidencia para reducir accidentes, establecer prioridades de intervención y evaluar el impacto de medidas implementadas. Los tableros permiten monitoreo en tiempo real y sirven como soporte visual para la toma de decisiones estratégicas.

👉 ¿Querés explorarlo? Descargá el archivo desde el repositorio y abrilo en Power BI para navegar las visualizaciones interactivas.

Tecnologías: Power BI, Excel

Además, el análisis exploratorio se desarrolló con Python y herramientas como Pandas, Seaborn, Matplotlib y Scikit-learn.
